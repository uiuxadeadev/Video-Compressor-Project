# UDP Chat Application

このプロジェクトは、複数のクライアントがUDPを使用してメッセージを送受信できるチャットアプリケーションです。
サーバーはクライアントからのメッセージを受信し、それを他のクライアントに中継します。

## 構成

- **server.py**: メッセージを受信し、他のクライアントにリレーするUDPサーバー
- **client.py**: サーバーに接続し、メッセージを送受信するUDPクライアント

## 必要条件

- Python 3.x
- socketモジュール（標準ライブラリに含まれています）

## インストール

このプロジェクトをクローンまたはダウンロードしてください：

```bash
git clone <repository-url>
cd <repository-directory>
```

## 使用方法

### サーバーの起動

1. `server.py` を実行してサーバーを起動します。

```bash
python server.py
```

サーバーは `0.0.0.0:9001` で待機します。

### クライアントの起動

1. 別のターミナルで `client.py` を実行してクライアントを起動します。

```bash
python client.py
```

2. サーバーのアドレスを入力します（デフォルトは `127.0.0.1`）。
3. ユーザー名を入力します。
4. メッセージを入力して送信します。

### クライアントの例

複数のクライアントを起動して、それぞれ異なるユーザー名を入力してください。

- クライアント1（`alice`）が `Hello` を送信
- クライアント2（`bob`）が `Hey` を送信

クライアント1のターミナルにクライアント2のメッセージが表示され、逆も同様です。

## コードの説明

### server.py

- **クライアント接続管理**: 各クライアントの接続を管理し、タイムアウトしたクライアントを削除します。
- **メッセージの処理**: 受信したメッセージを解析し、他のクライアントにリレーします。

### client.py

- **メッセージ送信**: サーバーにメッセージを送信するための関数を提供します。
- **メッセージ受信**: サーバーからのメッセージを受信し、表示します。

## 注意事項

- UDPプロトコルを使用しているため、メッセージの順序や完全性は保証されません。
- サーバーはシングルスレッドで動作するため、同時に大量のクライアントからの接続を処理するのには向いていません。
